# Hyprland Config
This is My Hyprland Config, how to use it

## 🚀 How to Install

> [!WARNING]
> I Recomend backup first on your ~/.config, to avoid lost data

### 1. Install AUR
first install aur, i using yay
```bash
# Clone dan install yay
sudo pacman -S --needed base-devel git
git clone [https://aur.archlinux.org/yay.git](https://aur.archlinux.org/yay.git)
cd yay
makepkg -si
cd .. && rm -rf yay
```

### 2. Clone This Repo
Second, clone this repo :
```bash
git clone https://github.com/ShirayukiHimeji/MyConfigHyprland.git
```
